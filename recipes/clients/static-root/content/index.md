# Recipes Static Root

This is a static root client built with VitePress.

It is configured to reuse recipes app visual configuration:

- Vuetify plugin styles from `recipes/clients/build/overrides.scss`
- Shared Vuetify options from `recipes/clients/common/vuetify-config.ts`

Go to [Vuetify Demo](/vuetify-demo) to see shared components/styles in action.
